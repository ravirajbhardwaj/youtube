### Installation


```bash
git clone git@github.com:ravirajbhardwaj/youtube.git
cd youtube
```

### server

```bash
cd server
pnpm install
cp .env.example .env
pnpm dev
```

### web

```bash
cd web
pnpm install
cp .env.example .env
pnpm dev
```
